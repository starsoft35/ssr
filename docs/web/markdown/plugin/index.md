## 社区插件

待更新。。。

如果你开发了一个新的服务端插件或客户端插件可提 [PR](https://github.com/zhangyuang/ssr/pulls) 修改本文档

目前客户端插件可能需要 `angular` 插件
服务端插件可以考虑新增 [plugin-daruk](https://github.com/darukjs/daruk)

不建议新增纯粹的 `koa` `express` 插件。可被已有插件代替
